中文 → 日文 即時翻譯朗讀工具

可手打中文、中文朗讀、按住說話轉文字、翻譯成日文、播放日文，並支援再次播放。

建議使用 Chrome / Edge 測試語音辨識
按住「按住說話」開始辨識，放開後停止。桌機可用滑鼠，手機可用觸控。
目前內建兩種模式：
1. 示範模式：用簡單字典做翻譯，不需金鑰，但翻譯有限。
2. API 模式：接翻譯 API，可做真正較完整的翻譯。
系統就緒，請輸入中文或按住按鈕開始說話。

API 模式說明

這份 HTML 已經預留 translateWithApi()，你可以改成自己的翻譯服務，例如：
1. 你自己的 Python / Node.js 後端
2. 已購買金鑰的翻譯 API
3. 本地端翻譯服務
